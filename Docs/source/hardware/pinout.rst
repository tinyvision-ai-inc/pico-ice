   GPIO4     IOT_38b               │ Pico-FPGA bus
   GPIO5     IOT_36b               │

   GPIO19    PICO_PMOD_B4          │

   GPIO20    SRAM_SSN              │ CS of the FPGA SRAM chip

   GPIO24    PICO_LED_B            │
